# Gap Analysis Prompt

Fill in the bracketed sections below, then paste the entire prompt into any LLM.

---

I need you to analyze my resume and tell me what I am missing technically — specifically what domain or project I should build to strengthen my resume for the role I am targeting.

**Target role:** [Data Engineer / ML Engineer / Backend Engineer / Full-Stack Engineer / Data Scientist / DevOps Engineer / Frontend Engineer / Mobile Engineer]

**Years of experience:** [0–1 / 1–3 / 3–7 / 7+]

**Background:** [Student / Career changer / Experienced in the same domain]

**Current projects on my resume:**
1. [Project name — one sentence describing what it does + tech stack + any measurable impact or scale]
2. [...]
3. [...] (write "No projects yet" if applicable)

**Current skills on my resume:**
[Languages, Frameworks, Tools, Platforms — copy-paste from your resume]

**Job description keywords (optional):** [If you have a specific job posting, list the required tools or keywords here]

---

Based on the information above, I need you to answer three things:

1. **The missing proof point:** What specific signal is absent from my resume that every recruiter looks for in this role?

2. **The recommended project:** What exactly should I build to close this gap? Be specific:
   - A clear project name (not just a tech stack)
   - The recommended tech stack
   - Scope that is achievable in 1–2 weeks
   - What the deliverable looks like (deployed link, public notebook, installable APK, live API, etc.)

3. **One sentence:** Why does this specific gap hurt my chances with a recruiter for this role?
